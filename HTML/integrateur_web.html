Web Integrator

HIS TASK

The web integrator is responsible for the creation of web sites using the elements outlined by the project manager or the webmaster : texts, pictures, videos, table...using the appropriate tools and software while meeting client expectations, deadlines and allocated budgets. The Integrator’s job begins when the project design phase is complete and it is time to implement the design. The web integrator coordinates the work of all those involved in the web integration project. Their role is to ensure that the work is consistent with the project.
